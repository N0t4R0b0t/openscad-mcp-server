# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [0.1.0] - 2024-04-26

### Added

- Initial release of OpenSCAD MCP Server
- `write_scad` tool for writing OpenSCAD code to files
- `render_preview` tool for generating PNG previews
- `export_stl` tool for exporting STL files for 3D printing
- Local stdio-based MCP server with no authentication
- Support for Claude Desktop integration
- Comprehensive README with usage examples
- Contributing guidelines
- AGPL-3.0 license

### Features

- Text-based OpenSCAD code generation via Claude
- Real-time preview rendering
- Direct STL export for 3D printers
- Local-only operation (no cloud dependencies)
- Fast iteration workflow
